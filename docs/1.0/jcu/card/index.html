Overlay Cards

Overlay Cards are customised Bootstrap Cards with specialised styling and animation. They rely upon Overlay, another JCU component, for colouring and design of short content that’s placed on top of something else (typically an image).

Refer to Card from Bootstrap’s core for full documentation about these and other aspects to Cards.

Examples

The .jcu-overlay-card modifier adjusts positioning and adds transition effects, as the examples show. The .jcu-overlay classes, sourced from Overlay, style and colourise just the overlay inside the cards.

Heads up! Any images placed within a .card must be configured to be responsive via .img-fluid or else it will overflow outside if too wide for the card. This is especially important because card widths may be inherited from their parent container (such as a column) and thus may be different sizes on different devices.

‘Fixed’ variation

By using the .jcu-overlay-card--fixed modifier together with an Overlay Card, any animations and interactivity with the card are suppressed.

Fixed overlay, no animation
This is a short, optional subtitle.
 
<div class="card jcu-overlay-card jcu-overlay-card--fixed">
  <a href="https://jcu.edu.au">
    <img class="card-img img-fluid" src="../images/card.jpg" alt="Stingray on ocean floor">
    <div class="card-img-overlay jcu-overlay jcu-bg--plain-border">
      <div class="card-title">Fixed overlay, no animation</div>
      <div class="card-subtitle">This is a short, optional subtitle.</div>
    </div>
  </a>
</div>

Sizes

Card sizing is dictated by the image or other background content that you place inside the card and the container in which the card is placed, such as the grid or layout system. For empty cards, such as those with just a plain background colour, you get a default minimum height and for larger sizes, specify an inline style for min-height.

Yellow, 50% opacity, standard height
Blue, 90% opacity; 10rem min-height
